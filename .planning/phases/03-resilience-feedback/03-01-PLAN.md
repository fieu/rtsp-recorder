Key requirements:
- ValidateRTSP(url string, timeout time.Duration) error — performs DESCRIBE request
- IsStreamAccessible(url string) bool — quick check wrapper
- DESCRIBE request: 10-second timeout (D-36)
- "Accessible" criteria: 200 OK with valid SDP response (D-37)
- Fail fast with clear error message if DESCRIBE fails (D-38)

Implementation approach:
- Use net.DialTimeout for TCP connection to RTSP server
- Send DESCRIBE request formatted per RTSP 1.0 spec
- Parse response for 200 OK and Content-Type: application/sdp
- Return descriptive errors:
  - "[ERROR] Cannot connect to RTSP server. Check IP address and port."
  - "[ERROR] Stream not found. Verify the RTSP URL path."
  - "[ERROR] Authentication required. Check username/password in URL."

Parse URL to extract host:port from rtsp:// scheme. Handle edge cases:
- Missing scheme prefix
- Invalid URL format
- Connection refused
- 401/403 authentication errors
- 404 not found
- Timeout (camera offline)

Create internal/errors/classifier.go implementing error classification per D-39 through D-44.

Categories (D-41):
- NetworkError — Connection, timeout, route issues (retryable)
- AuthenticationError — 401, 403 (not retryable)
- StreamError — Invalid data, codec issues (not retryable)
- ConfigurationError — Invalid URL, missing fields (not retryable)
- FFmpegError — Internal ffmpeg failures (not retryable)

Error patterns to detect (D-40):
- `Connection refused` → NetworkError: "[ERROR] Cannot connect to camera. Check IP address and port."
- `404 Not Found` → StreamError: "[ERROR] Stream path not found. Verify the RTSP URL path."
- `Invalid data found when processing input` → StreamError: "[ERROR] Stream data invalid. Camera may be offline or incompatible."
- `No route to host` / `Network is unreachable` → NetworkError: "[ERROR] Network unreachable. Check network connectivity."
- `401 Unauthorized` / `403 Forbidden` → AuthenticationError: "[ERROR] Authentication required. Check username/password in URL."
- `Operation timed out` → NetworkError: "[ERROR] Connection timeout. Camera may be offline or behind firewall."

Functions to export:
```go
type ErrorCategory string

const (
    NetworkError       ErrorCategory = "network"
    AuthenticationError ErrorCategory = "auth"
    StreamError        ErrorCategory = "stream"
    ConfigurationError ErrorCategory = "config"
    FFmpegError        ErrorCategory = "ffmpeg"
)

type ClassifiedError struct {
    Category ErrorCategory
    Message  string
    Retryable bool
    Original error
}

func ClassifyError(stderr string, exitCode int) *ClassifiedError
func FormatErrorMessage(classified *ClassifiedError) string
func IsRetryable(category ErrorCategory) bool
```
