## API Mapping (Zap → Zerolog)
From analysis of current codebase:
- `*zap.Logger` type → `zerolog.Logger` (value type, not pointer)
- `logger.Info(msg, zap.String(k,v))` → `logger.Info().Str(k,v).Msg(msg)`
- `logger.Warn(msg, zap.String(k,v), zap.Error(err))` → `logger.Warn().Str(k,v).Err(err).Msg(msg)`
- `logger.Debug(msg, zap.String(k,v), zap.Bool(k,v))` → `logger.Debug().Str(k,v).Bool(k,v).Msg(msg)`
- `zap.String(k,v)` → `.Str(k,v)`
- `zap.Int(k,v)` → `.Int(k,v)`
- `zap.Duration(k,v)` → `.Dur(k,v)`
- `zap.Error(err)` → `.Err(err)`
- `zap.Bool(k,v)` → `.Bool(k,v)`
