Add StopManager to recorder/stop_conditions.go that coordinates multiple monitors per D-17:

1. StopManager struct:
   ```go
   type StopManager struct {
       mu       sync.Mutex
       monitors []Monitor
       stop     chan StopReason
       ctx      context.Context
       cancel   context.CancelFunc
   }
   
   type StopReason struct {
       Name string // Which monitor triggered: "signal", "duration", "file_size"
       Desc string // Human-readable description
   }
   ```

2. NewStopManager() *StopManager:
   - ctx, cancel := context.WithCancel(context.Background())
   - Return &StopManager{
       stop:   make(chan StopReason, 1), // Buffered to prevent blocking
       ctx:    ctx,
       cancel: cancel,
   }

3. (sm *StopManager) AddMonitor(m Monitor):
   - sm.monitors = append(sm.monitors, m)

4. (sm *StopManager) Start():
   - For each monitor: m.Start(sm.ctx)
   - Start coordination goroutine:
     ```go
     go func() {
         // Create select cases dynamically
         cases := make([]reflect.SelectCase, len(sm.monitors)+1)
         cases[0] = reflect.SelectCase{Dir: reflect.SelectRecv, Chan: reflect.ValueOf(sm.ctx.Done())}
         
         for i, m := range sm.monitors {
             cases[i+1] = reflect.SelectCase{
                 Dir:  reflect.SelectRecv,
                 Chan: reflect.ValueOf(m.Wait()),
             }
         }
         
         chosen, _, _ := reflect.Select(cases)
         
         if chosen == 0 {
             // Context cancelled externally
             sm.stop <- StopReason{Name: "context", Desc: "Recording cancelled"}
         } else {
             // One of the monitors triggered
             monitor := sm.monitors[chosen-1]
             sm.stop <- StopReason{Name: monitor.Name(), Desc: sm.describeReason(monitor.Name())}
         }
         
         sm.cancel() // Cancel context to stop other monitors
     }()
     ```

5. Helper describeReason(name string) string:
   - switch name:
     - "signal": return "Interrupted by user (Ctrl+C)"
     - "duration": return "Duration limit reached"
     - "file_size": return "File size limit reached"
     - default: return "Stop condition triggered"

6. (sm *StopManager) Wait() <-chan StopReason:
   - Return sm.stop

7. (sm *StopManager) Stop():
   - sm.cancel()

8. (sm *StopManager) Context() context.Context:
   - Return sm.ctx

Note: Uses reflection for dynamic select, or can use multiple goroutines per monitor. Choose simpler approach.

Alternative simpler implementation using multiple goroutines (more readable, less reflection magic):
```go
func (sm *StopManager) Start() {
    var wg sync.WaitGroup
    
    for _, m := range sm.monitors {
        wg.Add(1)
        go func(mon Monitor) {
            defer wg.Done()
            select {
            case <-mon.Wait():
                select {
                case sm.stop <- StopReason{Name: mon.Name(), Desc: sm.describeReason(mon.Name())}:
                    sm.cancel() // Signal other monitors
                default:
                    // Another monitor already triggered, ignore
                }
            case <-sm.ctx.Done():
                return
            }
        }(m)
    }
    
    // Cleanup goroutine
    go func() {
        wg.Wait()
        close(sm.stop)
    }()
}
```

This is cleaner — use this pattern.
